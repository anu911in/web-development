Some notes on italics and bold

Use "em" tag for italics as the "em" element represents stress emphasis of its contents

"i" element represents text that is set off from the normal prose, such a foreign word, fictional character thoughts, or when the text refers to the definition of a word instead of representing its semantic meaning.

Use "strong" for bold as element indicates that its contents have strong importance, seriousness, or urgency. Browsers typically render the contents in bold type.

This "b" element helps in bold face. However, you should use the CSS "font-weight" property to create boldface text, or the "strong" element to indicate that text is of special importance.
